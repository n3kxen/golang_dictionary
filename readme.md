# GO program to add words and find synonyms 

The Program processes the following queries in a loop until the query `done` is entered:

## Functionality

- `add word1 word2` Input two words, `word1` and `word2`, and add them to the dictionary
- `count word` Input a word, `word`, and output the number of synonyms for the entered word
- `check word1 word2` Input two words and check if the entered words are synonyms (output the message "Yes" or "No")
